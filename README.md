## Uitwerking
Alles is gemaakt met HTML, CSS en JavaScript. De basale vormen zijn via clip-paths gemaakt in illustrator en omgezet naar SVG. Ik kreeg het alleen niet helemaal voor elkaar met relatieve clip-paths in CSS-only zonder SVG tag in HTML helaas.. Misschien de volgende keer eens wat verder in verdiepen. Het werkte wel met simpele vormen, maar niet met de complexere paths die ik heb gemaakt. Mogelijk dat het daar fout gaat :D
